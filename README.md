# PharmaShield Native Landing

PharmaShield Native Landing is a static frontend project for presenting the PharmaShield brand and user-facing pages for a blockchain-based drug authentication and supply-chain concept.

## Included Pages

- `index.html` for the main landing page
- `authentication.html` for authentication-related UI
- `style.css` for the landing-page styles
- `authentication.css` for the authentication-page styles
- `app.js` for mobile-menu interaction and small UI behaviors

## Tech Stack

- HTML
- CSS
- Vanilla JavaScript
- Static assets and browser-based UI flows

## Key UI Elements

- Fixed navigation bar
- Responsive branding and logo treatment
- Mobile-menu toggle behavior
- Styled authentication and landing sections
- Custom color palette and typography

## Getting Started

1. Open the project folder in your editor.
2. Launch `index.html` directly in a browser, or serve the folder with any static file server.
3. Edit `style.css`, `authentication.css`, and `app.js` to customize the experience.

## Notes

- This repository is frontend-only, so no backend build step is required.
- The project is a useful companion landing page for the broader PharmaShield system repos.
